# yt-dlp | youtube Downloader

so basically first setup the venv by:

```sh
python3 -m venv myvenv
source myenv/bin/activate
```

install the dependencies:

```sh
pip install moviepy yt_dlp
```

to run a gui:

```sh
python3 clipper.py
```
